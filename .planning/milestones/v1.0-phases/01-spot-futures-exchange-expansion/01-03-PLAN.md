**Per-Exchange Research (from 01-RESEARCH.md):**

Gate.io:
- Account model: Unified account (similar to Bybit)
- Auto-borrow: `auto_borrow=true` / `auto_repay=true` on spot order with `account=unified`
- Transfer: No-op (unified account)
- Blackout: None known
- Order placement: `/spot/orders` with `account: unified`
- Market BUY: Amount is in quote currency (USDT) when side=buy
- Unified mode detection: `DetectUnifiedMode()` exists in adapter
- Response format: JSON body (vs query string for other exchanges)

OKX (ACCOUNT-LEVEL autoLoan):
- Account model: Unified account, but manual borrow/repay by default
- Auto-borrow: Account-level `autoLoan` setting (NOT per-order). Set via `POST /api/v5/account/set-auto-loan` with `{"autoLoan": true}`
- Once autoLoan=true is configured, OKX automatically borrows when selling assets you don't hold and auto-repays when buying back -- this IS exchange-native, just configured at account level rather than per-order
- Transfer: No-op (unified account)
- Borrow endpoint (manual fallback): `/api/v5/account/spot-manual-borrow-repay`
- Contract size: OKX uses `ctVal` (contract value multiplier) for futures
- Symbol format: Spot = `BTC-USDT`, Futures = `BTC-USDT-SWAP`
- PlaceSpotMarginOrder uses `tdMode=cross` for cross-margin spot orders
