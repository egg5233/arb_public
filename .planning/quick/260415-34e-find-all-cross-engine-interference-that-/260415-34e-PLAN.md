## The Core Problem

Exchange-level API calls return ALL positions/balances on an exchange, regardless of which engine placed them:
- `getExchangePositionSize()` calls `exch.GetPosition()` which returns the TOTAL exchange position for a symbol
- `exch.GetAllPositions()` returns ALL futures positions on the exchange
- `exch.GetFuturesBalance()` returns the total margin balance including both engines' exposure
- WebSocket fill callbacks fire for ANY order on the account, not just the subscribing engine's orders

When both engines have positions on the same exchange (and potentially the same symbol), any logic that assumes "exchange position = my engine's position" will produce wrong results.
