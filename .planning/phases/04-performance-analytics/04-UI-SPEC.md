# Phase 4 -- UI Design Contract

> Visual and interaction contract for the Performance Analytics phase. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none |
| Preset | not applicable |
| Component library | none (raw Tailwind CSS v4 utility classes) |
| Icon library | Unicode characters (existing pattern -- sidebar uses Unicode glyphs) |
| Font | System default (`font-sans` -- no custom font loaded) |

**Rationale:** The existing dashboard uses raw Tailwind utilities without shadcn or any component library. Introducing shadcn mid-stream for one phase would create inconsistency. The Analytics page and History drill-down follow established patterns.

## Typography

| Role | Size | Weight | Line Height | Tailwind Class |
|------|------|--------|-------------|----------------|
| Body | 14px | 400 (normal) | 1.5 | `text-sm` |
| Label | 12px | 400 (normal) | 1.4 | `text-xs` |
| Heading | 20px | 700 (bold) | 1.2 | `text-xl font-bold` |
| Subheading | 16px | 700 (bold) | 1.3 | `text-base font-bold` |

**Weights:** Exactly 2 weights used throughout this phase -- 400 (normal) for body text, labels, data values, and table content; 700 (bold) for page headings, section headings, and stat card values.

**Source:** Extracted from existing patterns. Page headings use `text-xl font-bold` (History.tsx:52, Overview.tsx:103). Body text uses `text-sm`. Labels and table headers use `text-xs` or `text-sm text-gray-400`. Stat card values use `font-bold`.

## Page Layouts

### Analytics Page (`/analytics`)

```
+--------------------------------------------------+
| Analytics                        [7d][30d][90d][All] |  <-- page heading + TimeRangeSelector
+--------------------------------------------------+
|                                                    |
| [Cumulative PnL Chart - full width]               |  <-- PnLChart (height: 400px)
| [Brush selector bar at bottom]                     |
|                                                    |
+--------------------------------------------------+
|                                                    |
| Strategy Comparison          Exchange Performance  |  <-- 2-column grid on desktop
| [Bar chart: PP vs SF PnL]   [Table + bars]        |     single column on mobile
| [Summary stats below]       [Profit, APR, Win%]   |
|                                                    |
+--------------------------------------------------+
| Summary Stats Row (4 cards):                       |  <-- grid-cols-2 md:grid-cols-4
| [Total PnL] [APR] [Win Rate] [Avg Hold Time]     |
+--------------------------------------------------+
```

**Layout rules:**
- Page heading: `text-xl font-bold text-gray-100` (matches History, Overview pattern)
- TimeRangeSelector: aligned right on same row as heading, using `flex items-center justify-between`
- Main chart: full width, `bg-gray-900 border border-gray-800 rounded-lg p-4`, height 400px
- Strategy + Exchange: `grid grid-cols-1 lg:grid-cols-2 gap-8` below main chart
- Summary stats: `grid grid-cols-2 md:grid-cols-4 gap-4` at bottom
- Section gaps: `space-y-8` between major sections (chart, comparison row, stats)
- Mobile: all sections stack vertically, charts maintain min-height 300px

### History Page Drill-Down (enhanced)

```
+--------------------------------------------------+
| [existing table row]                     [expand] |  <-- click row to expand
+--------------------------------------------------+
| PnL Breakdown (expanded, colSpan full):           |
|                                                    |
| Entry Fees    Exit Fees     Funding      Basis    |  <-- grid-cols-2 md:grid-cols-4
| -$1.23        -$0.98        +$4.56      +$0.12   |
|                                                    |
| Borrow Cost   Slippage      Net PnL     APR      |
| -$0.34        -$0.02        +$2.11      18.4%    |
|                                                    |
+--------------------------------------------------+
```

**Drill-down rules:**
- Expanded row: `bg-gray-800/50` background (subtle differentiation from table rows)
- Grid: `grid grid-cols-2 md:grid-cols-4 gap-4 text-sm`
- Each cell: label in `text-gray-500 text-xs` above value
- Values: green for positive, red for negative, gray for zero/N/A
- Missing decomposition fields (pre-v0.27.0 positions): show "-" in `text-gray-500`
- Borrow cost field: only visible for spot-futures positions (hide for perp-perp)
- Click same row again to collapse

## Chart Specifications

### Cumulative PnL Chart (PnLChart)

| Property | Value |
|----------|-------|
| Chart type | `LineChart` (Recharts) |
| Height | 400px (desktop), 300px (mobile) |
| Background | `bg-gray-900` card with `border-gray-800` |
| Line color (combined) | `#34D399` (green-400) |
| Line color (perp-perp) | `#60A5FA` (blue-400) -- visible when strategy filter active |
| Line color (spot-futures) | `#C084FC` (purple-400) -- visible when strategy filter active |
| Grid | `strokeDasharray="3 3"` with `stroke="#374151"` (gray-700) |
| X-axis | Date labels, `stroke="#9CA3AF"` (gray-400), format: `MM/DD` or `MM/DD HH:mm` for < 7d |
| Y-axis | Dollar values, `stroke="#9CA3AF"`, format: `$XXX` |
| Tooltip background | `backgroundColor: '#1F2937'` (gray-800) with `border: '1px solid #374151'` (gray-700) |
| Tooltip label | Full datetime: `YYYY-MM-DD HH:mm` |
| Tooltip value | `$X.XX` with 2 decimal places |
| Brush | Height 30px, `stroke="#4B5563"` (gray-600), enables drag-to-zoom |
| Dot | `dot={false}` (no individual data point markers -- line only) |
| Animation | Default Recharts transition (300ms ease) |
| Responsive | `<ResponsiveContainer width="100%" height={400}>` |

### Strategy Comparison Chart (StrategyComparison)

| Property | Value |
|----------|-------|
| Chart type | `BarChart` (Recharts) grouped bars |
| Height | 300px |
| Bar colors | Blue (`#60A5FA`) for perp-perp, Purple (`#C084FC`) for spot-futures |
| Bar radius | `radius={[4, 4, 0, 0]}` (rounded top corners) |
| X-axis | Time periods (days or weeks depending on range) |
| Y-axis | Dollar values |
| Tooltip | Same styling as PnLChart |
| Legend | Below chart, `text-xs text-gray-400`, inline items with colored dot indicators |

### Exchange Metrics Table (ExchangeMetrics)

| Property | Value |
|----------|-------|
| Layout | Table with inline progress bars |
| Columns | Exchange, Profit, APR, Win Rate, Trades, Avg Slippage |
| Profit column | Colored value (green/red) + horizontal bar indicator |
| Bar max width | 120px, background `bg-gray-700`, fill uses exchange-proportional width |
| Bar fill color | `bg-green-500/60` for profit, `bg-red-500/60` for loss |
| Sort | Default sorted by profit descending |
| Row hover | `hover:bg-gray-800/50` |

## Interaction Contracts

### Analytics Page Interactions

| Interaction | Behavior |
|-------------|----------|
| Time range change | All charts re-query API with new `from`/`to` params; show loading state during fetch |
| Chart brush drag | Zooms the PnL chart to selected sub-range; does NOT affect other charts |
| Chart tooltip hover | Shows data point details; tooltip follows cursor within chart bounds |
| Exchange row click | No drill-down in this phase (future enhancement) |
| Page load | Fetch analytics data with default 30d range; show skeleton/loading state while loading |

### History Page Interactions

| Interaction | Behavior |
|-------------|----------|
| Row click | Toggles PnL breakdown expansion for that row (accordion -- only one row expanded at a time) |
| Expanded row | Shows decomposed PnL grid below the table row |
| Missing data | Fields unavailable for pre-v0.27.0 positions display "-" |

### Loading States

| State | Visual |
|-------|--------|
| Analytics page loading | Skeleton placeholder: `bg-gray-800 animate-pulse rounded-lg` matching chart dimensions |
| Chart data loading | Chart container visible with centered `text-sm text-gray-500` "Loading..." text |
| History drill-down loading | Inline `text-xs text-gray-500` "Loading..." in expanded row area |

## Accessibility

| Requirement | Implementation |
|-------------|----------------|
| Chart keyboard nav | Recharts provides built-in keyboard accessibility for tooltips |
| Color contrast | All text meets WCAG AA (gray-400 on gray-900 = 4.64:1, gray-100 on gray-950 = 15.3:1) |
| Profit/loss not color-only | Values include `$` sign and `+`/`-` prefix in addition to green/red coloring |
| Screen reader | Chart data available in tabular format below charts for screen readers (SR-only table) |
| Expandable rows | Use `aria-expanded` on clickable History rows |
| Time range buttons | Use `aria-pressed` for active state |
