**Part B: Native Discovery Scanner (internal/spotengine/discovery.go)**

Add a new method `pollLoris()` on SpotEngine that replicates the pattern from `internal/discovery/scanner.go:220-242`:
- HTTP GET to `https://api.loris.tools/funding`
- Decode into `models.LorisResponse`
- Return `(*models.LorisResponse, error)`

Add a new method `runNativeDiscoveryScan() []SpotArbOpportunity`:
1. Call `e.pollLoris()`. On error, log and return `e.runCoinGlassFallback()`.
2. Build allowed exchanges set (reuse existing pattern from line 98-109).
3. For each symbol in `loris.Symbols`:
   - Construct `symbol = strings.ToUpper(baseSym) + "USDT"`
   - For each exchange in `e.spotMargin`:
     - Skip if not in allowed exchanges (when configured)
     - Look up `loris.FundingRates[exchName][baseSym]` -- skip if missing
     - Normalize: `bpsPerHour = rawRate / 8.0`, `fundingAPR = bpsPerHour * 8760.0 / 10000.0`
     - Look up `loris.FundingIntervals[exchName][baseSym]` for settlement info (store in opp if available)
     - **Dir A** (borrow_sell_long) -- when `fundingAPR > 0`:
       - Fetch borrow rate via `e.getCachedBorrowRate(exchName, baseSym, smExch)`
       - If borrow rate unavailable, skip Dir A for this symbol+exchange
       - `borrowAPR = rate.YearlyRate` (already decimal APR)
       - `feeAPR = spotFees[exchName] * 4 * (365.0 / assumedHoldDays)` (4 legs round trip)
       - `netAPR = fundingAPR - borrowAPR - feeAPR`
       - Create `SpotArbOpportunity{..., Direction: "borrow_sell_long", Source: "native", ...}`
       - Apply filter: if `netAPR < e.cfg.SpotFuturesMinNetYieldAPR`, set `FilterStatus = "below_min_yield"`
       - If `borrowAPR > e.cfg.SpotFuturesMaxBorrowAPR && e.cfg.SpotFuturesMaxBorrowAPR > 0`, set `FilterStatus = "borrow_too_high"`
     - **Dir B** (buy_spot_short) -- when `fundingAPR > 0`:
       - `borrowAPR = 0` (no borrow for Dir B)
       - Same feeAPR, netAPR calculation
       - Create `SpotArbOpportunity{..., Direction: "buy_spot_short", Source: "native", ...}`
       - Apply same filters
4. Sort by `NetAPR` descending.
5. Return all opportunities (both filtered and unfiltered, per existing convention).

Rename existing `runDiscoveryScan()` body to `runCoinGlassFallback() []SpotArbOpportunity`. This preserves the existing CoinGlass logic intact.

Modify `runDiscoveryScan()` to be the router:
```go
func (e *SpotEngine) runDiscoveryScan() []SpotArbOpportunity {
    if e.cfg.SpotFuturesNativeScannerEnabled {
        return e.runNativeDiscoveryScan()
    }
```

Per D-02, CoinGlass is fallback only. Per D-03, net yield = funding - borrow - fees.

CRITICAL -- Loris rate normalization (per project_loris_normalization.md and RESEARCH Pitfall 1):
- Loris returns 8h-equivalent basis points
- Divide by 8 to get bps/hour
- Multiply by 8760 (hours/year) and divide by 10000 (bps to decimal) to get APR
- Formula: `fundingAPR = (rawRate / 8.0) * 8760.0 / 10000.0`

Per Pitfall 2 (RESEARCH): Generate BOTH Dir A and Dir B as separate opportunities for each symbol+exchange when funding is positive. Let ranking sort naturally.
