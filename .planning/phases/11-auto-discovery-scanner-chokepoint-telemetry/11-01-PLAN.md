<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| operator → config.json | Operator edits config.json directly (universe, denylist, score floors). File is the trust boundary; no in-product UI mutates these in Phase 11. |
| config.go LoadJSON → in-memory `*Config` | Untrusted JSON crosses validation gate; invalid entries must reject at load, not at runtime. |
| pricegaptrader.RegistryReader → consumer packages | Compile-time read-only contract; mutator drift is the threat. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-11-01 | Tampering | Operator typo `PriceGapDiscoveryUniverse: ["btc-usdt"]` reaches scanner as malformed exchange call | mitigate | Validate on load with regex `^[A-Z0-9]{2,20}USDT$`; reject before LoadJSON returns. Scanner cannot run with invalid universe (no fallback). |
| T-11-02 | Tampering | Operator typo `PriceGapAutoPromoteScore: 0` causes Phase-12 to promote everything | mitigate | Server-side floor 50 enforced in `validatePriceGapDiscovery`; reject load below floor. Floor is also documented in CONTEXT D + RESEARCH §Security V5. |
| T-11-03 | Tampering | Future commit silently adds `Add`/`Update` to RegistryReader, breaking compile-time read-only | mitigate | `TestRegistryReader_NoMutatorMethods` greps the interface block in CI; any mutator regex hit fails the test. |
| T-11-04 | Information disclosure | Validation error messages echo back arbitrary user input from config.json | accept | Config.json is operator-owned (read by the bot operator only); echo is intentional for debugging; no third party reads these errors. |
| T-11-05 | Denial of service | Universe of 1000 symbols × 6 exchanges × per-tick I/O DOS the bot | mitigate | `len(PriceGapDiscoveryUniverse) ≤ 20` validated on load — hard bound at the schema layer (D-05). |
| T-11-06 | Tampering | `PriceGapDiscoveryIntervalSec=1` triggers per-second scans, hammering exchange WS caches | mitigate | Floor of 60 enforced at validation; rejected below. |
| T-11-07 | Repudiation | Module-boundary violation: pricegaptrader/registry_reader.go imports internal/engine | mitigate | Acceptance criterion greps for `internal/engine|internal/spotengine` import; CI fails on hit. |
</threat_model>
