    Public method RunCycle(ctx context.Context, now time.Time):
      1. If cfg.PriceGapDiscoveryEnabled is false: telemetry.WriteEnabledFlag(ctx, false); return.
      2. telemetry.WriteEnabledFlag(ctx, true).
      3. Snapshot universe + denylist + threshold + depthFloor + stalenessSec under cfg.RLock().
      4. Snapshot promoted candidates as map[string]bool keyed by "sym:long:short:direction" (read-only via s.registry.List()).
      5. For each symbol in universe:
           a. listings := listingExchanges(sym) — exch IDs where Exchange.GetBBO(sym) returns ok=true.
           b. If len(listings) < 2: telemetry.WriteSymbolNoCrossPair(ctx); continue.
           c. For each (longExch, shortExch) ordered pair where longExch != shortExch:
                If consecErr ≥ 5: telemetry.WriteCycleFailed(ctx); summary.Errors++; goto persist.
                rec := scanPair(...); accumulate into summary.
      6. summary.CompletedAt = nowFunc().Unix(); telemetry.WriteCycle(ctx, summary).

    scanPair gate evaluation order (short-circuit cheapest first):
      Gate 1: Denylist match (sym OR sym@exch on either leg) → ReasonDenylist.
      Gate 2: Bybit blackout (longExch==bybit OR shortExch==bybit during :04-:05:30) → ReasonBybitBlackout.
      Gate 3: BBO sample both legs (sampleLeg helper). On error → ReasonSampleError; consecErr++.
      Gate 4: BBO freshness <90s on both legs (mirror detector.go:194-201) → ReasonStaleBBO.
      Gate 5: Depth probe ≥ floor (min of legL.depthClearable, legR.depthClearable) → ReasonInsufficientDepth.
      Gate 6: ≥4-bar persistence via barRing.allExceedDirected(threshold, models.PriceGapDirectionBidirectional) → ReasonInsufficientPersistence.
      All gates pass: compute magnitude = clamp(0.5 * spreadExcessNorm + 0.3 * depthScore + 0.2 * max(fundingAlignment, 0), 0, 1) * 100. Round to int. WhyRejected = ReasonAccepted.
